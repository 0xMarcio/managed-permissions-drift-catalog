This page lists the IAM roles and permissions for BigQuery Omni. To search through all roles and permissions, see the role and permission index.

BigQuery Omni roles

BigQuery Omni offers the following service agent roles. Service agent roles should only be granted to service agents.

Role Permissions

BigQuery Omni Service Agent

(roles/bigqueryomni.serviceAgent)

Gives BigQuery Omni access to tables in user projects.

Warning: Do not grant service agent roles to any principals except service agents.

bigquery.jobs.create

bigquery.tables.updateData

BigQuery Omni permissions

There are no IAM permissions for this service.
